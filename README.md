# PvP Battleground Faction Icon 🟥🟦

A World of Warcraft addon that displays a faction icon when you're in a battleground, helping you quickly identify whether you're fighting for the Horde or Alliance.

## ✨ Features

- **Faction Icon Display** – Shows your faction's icon (Horde or Alliance) when you enter a battleground
- **Edit Mode Integration** – Position and configure the icon through WoW's built-in Edit Mode
- **Per-Layout Settings** – Size and border settings are stored independently for each Edit Mode layout
- **Adjustable Size** – Resize the icon square via a slider in the Edit Mode settings panel
- **Faction-Colored Border** – Optional colored border matching your faction (red for Horde, blue for Alliance), toggled via a checkbox in Edit Mode
- **Persistent Settings** – Position, size, and border preference are saved per layout between sessions

## 📦 Installation

1. Download the addon
2. Extract to your `World of Warcraft\_retail_\Interface\AddOns\` folder
3. Restart World of Warcraft or reload your UI with `/reload`

## ⚙️ Usage

The icon automatically appears when you join a battleground.

### Edit Mode

Open Edit Mode (**Escape → Edit Mode**) to:

- **Drag** the icon to reposition it
- **Click** the icon to open its settings panel:
  - **Size** – Slider to set the icon square size (16–128 px, default 48)
  - **Show Border** – Checkbox to toggle the faction-colored border

All settings are saved per Edit Mode layout — different layouts can have different positions, sizes, and border visibility.

### Slash Commands

- `/pbfi` – Show usage info
- `/pbfi debug` – Toggle debug mode
